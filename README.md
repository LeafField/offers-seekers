# 株式会社FLAT様 技術課題

## env

- RESAS API
  VITE_API_KEY=RESAS APIのAPI KEYを入れてください。

## 開発開始日

2024年4月27日

## 実行コマンド

初めに

```
npm i
```

で開発環境をインストールしてください。

- devサーバー

```
npm run dev
```

- ビルド

```
npm run build
```

- 静的解析（Prettier、ESLint、TypeScript）

```
npm run check
```

- スタイルガイド

```
npm run storybook
```

- ユニットテスト

```
npm test
```

- E2Eテスト
  ビルド後に

```
npm run e2e
```
